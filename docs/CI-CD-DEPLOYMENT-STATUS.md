### 可能的失败点及解决方案

#### 1️⃣ TypeScript 类型错误

**症状**: `tsc --noEmit` 失败  
**解决**:

```bash
pnpm type-check
# 根据错误提示修复类型问题
```

#### 2️⃣ ESLint 检查失败

**症状**: `pnpm lint` 报错  
**解决**:

```bash
pnpm lint --fix
# 或临时忽略（CI 中已配置 continue-on-error）
```

#### 3️⃣ 单元测试失败

**症状**: Vitest 测试未通过  
**解决**:

```bash
pnpm test:run
# 检查失败的测试用例并修复
```

#### 4️⃣ 构建失败

**症状**: `next build` 报错  
**解决**:

```bash
pnpm build
# 检查构建日志中的具体错误
```

#### 5️⃣ 部署权限错误

**症状**: GitHub Pages 部署权限不足  
**解决**:

- Settings → Pages → Source 选择 "GitHub Actions"
- 确保 Workflow 有 `pages: write` 权限
